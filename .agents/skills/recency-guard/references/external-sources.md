# External Sources

> Read this file only when a source-evaluation, reasoning, or disclosure-policy
> judgment needs more background than the bundled rules provide. Fetch the
> smallest relevant URL; do not load every source.

The skill remains functional without network access. These links replace long
static explanations and examples that would otherwise bloat the prompt.

## Use Rules

- Fetch one URL for the specific ambiguity; return to the local workflow after
  extracting the needed principle.
- Use these pages for static method guidance only. Current claim evidence still
  comes from current official, audited, or otherwise authoritative sources.
- If a URL is unavailable, continue with the bundled rules and mark confidence
  down only when the missing background materially affects the judgment.
